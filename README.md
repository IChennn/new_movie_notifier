# new_movie_notifier

Notify by Line official account message when there is a new movie.

### Required

- repository secret
  - `LINE_CHANNEL_ACCESS_TOKEN`
    - Line developer console > messaging API channel > Messaging API > Channel access token 
  - `LINE_USER_ID`
    - Line developer console > messaging API channel > Basic settings > Your user ID

- enable write permission
  - repo > Actions > General > Workflow permission
